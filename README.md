# Expense Tracker 

An expense tracking application built with ASP.NET MVC, and Syncfusion components.

## Features

### Core Features
- **Dashboard**: Real-time financial overview with charts and summaries
- **Transaction Management**: Add, edit, delete, and categorize transactions
- **Category Management**: Customizable income and expense categories
- **Reporting**: Generate financial reports and analytics

### Advanced Features
- **User Authentication & Authorization**: Secure login with role-based access
- **Responsive Design**: Mobile-friendly interface
- **Real-time Updates**: Live dashboard updates

## Technology Stack

### Backend
- ASP.NET Core 8.0
- Entity Framework Core 8.0
- SqlServer

### Frontend
- Bootstrap 5.3
- Syncfusion EJ2 Components
- ECharts for Data Visualization
- Font Awesome Icons
- jQuery 3.7

### Security
- ASP.NET Core Identity
- JWT Authentication
- Role-based Authorization
- Password Hashing (PBKDF2)
- HTTPS Enforcement

## Getting Started

### Prerequisites
- .NET 8.0 SDK
